## ng2 + tsc + typings + unit tests

Always updated with the latest version of ng2 (rc2), typescript (1.8.10), typings (1.0.4) and jasmine (2.3).

### Install dev dependencies

```shell
$ npm i
```

### Install typings globally

```shell
$ npm i -g typings
```

### Install typings stuff

```shell
$ typings i
```

### Compile

```shell
$ npm run build
```

### Run

```shell
$ npm start
```

### Test

```shell
$ npm t
```

## Happy coding!
